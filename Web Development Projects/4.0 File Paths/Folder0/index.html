Note: 1. Absolute file paths always start with a forward slash (/) - e.g, /project/images/photo.png. 2. Relative file paths do not start with a forward slash (/) - e.g, images/photo.png or ../images/photo.png. 3. Special characters in file paths: - A single dot (.) refers to the current directory. - A double dot (..) refers to the parent directory. 4. File paths are case-sensitive on most web servers, so ensure the case matches exactly.
